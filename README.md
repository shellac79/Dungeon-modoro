# 🎮 던전모도로 (Dungeon-modoro)

## 1. 프로젝트 개요
본 프로젝트는 뽀모도로 타이머와 텍스트 RPG 요소를 결합한 데스크톱 전용 집중력 강화 소프트웨어입니다. '시간 통제(집중)'와 '자원 수집 및 전투(성장)'라는 두 가지 축을 연동하여 사용자의 자발적인 몰입을 유도합니다.

## 2. 실행 방법

본 프로젝트를 로컬 컴퓨터에서 실행하려면 아래의 과정을 순서대로 진행해 주세요. (사전에 Python이 설치되어 있어야 합니다.)

### 📥 1) 코드 다운로드 및 열기
1. 우측 상단의 초록색 **[ <> Code ]** 버튼을 눌러 **[Download ZIP]**을 클릭하여 코드를 다운로드하고 압축을 풉니다.
2. VS Code를 실행한 뒤, 상단 메뉴의 **[파일] -> [폴더 열기]**를 통해 압축을 푼 폴더를 통째로 열어줍니다. 
   *(🚨 주의: `main.py` 파일만 단독으로 열면 오디오 파일과 세이브 데이터를 불러올 수 없습니다! 반드시 폴더 전체를 열어주세요.)*

### ⚙️ 2) 필요 라이브러리(엔진) 설치
게임 구동을 위해 `pygame` 라이브러리 설치가 필요합니다. VS Code 하단의 **터미널(Terminal)** 창을 열고 아래 명령어를 복사하여 붙여넣은 뒤 엔터를 쳐주세요.

```bash
pip install pygame
```
🚀 3) 게임 실행
설치가 완료되면, 터미널에 아래 명령어를 입력하여 게임을 실행합니다. (또는 우측 상단의 ▷ 재생 버튼을 눌러도 됩니다.)
```bash
 python main.py
```
## 3. 팀원 구성 및 역할 분담
* **팀장 이승엽:** 프로젝트 총괄, `main.py` 게임 루프 설계, Git 레포지토리 관리
* **팀원 박준현 (Timer & Penalty):** 뽀모도로 타이머 및 딴짓 차단 시스템 구현
* **팀원 엄민욱 (Data & Upgrade):** 캐릭터 데이터 관리 및 스탯 강화 로직 설계
* **팀원 김현준 (Battle & UI):** 스테이지형 보스전 및 전투 체력바 UI 구현
* **팀원 권혁훈 (Timer & IMAGE):** 뽀모도로 타이머 및 소리, 이미지 추가
