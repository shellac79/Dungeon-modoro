# 🎮 던전모도로 (Dungeon-modoro)

## 1. 프로젝트 개요
본 프로젝트는 뽀모도로 타이머와 텍스트 RPG 요소를 결합한 데스크톱 전용 집중력 강화 소프트웨어입니다. '시간 통제(집중)'와 '자원 수집 및 전투(성장)'라는 두 가지 축을 연동하여 사용자의 자발적인 몰입을 유도합니다.

## 2. 실행 방법
\`\`\`bash
pip install pygame
python main.py
\`\`\`

## 3. 팀원 구성 및 역할 분담
* **팀장 (본인):** 프로젝트 총괄, `main.py` 게임 루프 설계, Git 레포지토리 관리
* **팀원 A (Timer & Penalty):** 뽀모도로 타이머 및 딴짓 차단 시스템 구현
* **팀원 B (Data & Upgrade):** 캐릭터 데이터 관리 및 스탯 강화 로직 설계
* **팀원 C (Battle & UI):** 스테이지형 보스전 및 전투 체력바 UI 구현
