# 🎯 던전모도로 (Dungeon-odoro) 시스템 전체 기능 명세서

## 1. 💡 핵심 아이디어 (Core Concept)
* **뽀모도로 타이머 + RPG:** 컴퓨터 작업 시 딴짓을 방지하고 집중력을 극대화하기 위한 데스크톱 전용 애플리케이션입니다.
* **메인 슬로건:** "시간을 통제하여 버티면 캐릭터가 성장한다."

## 2. 🔄 3단계 코어 루프 (Core System Loop)

### ⏱️ Phase 1: 집중과 파밍 (딴짓 철통 방어)
* **던전 탐험 모드:** 25분 타이머 작동 시간을 캐릭터의 '던전 탐험' 상태로 규정합니다.
* **백그라운드 파밍:** 시간이 흐름에 따라 내부 조건문이 작동하여 자원이 임시 인벤토리에 주기적으로 수집됩니다.
* 🚨 **핵심 페널티 (딴짓 방지):** 집중 시간 중 알트탭(Alt+Tab)을 사용하거나 타 앱으로 화면 포커스를 이탈하는 순간, 수집된 임시 재화는 즉시 전량 소멸됩니다.
* **완주 보상:** 목표 시간을 이탈 없이 완주해야만 임시 재화가 실제 계정에 확정 지급됩니다.

### 🛠️ Phase 2: 스탯 강화와 자원 관리
* **스탯 업그레이드:** 타이머 종료 후 주어지는 휴식 시간 동안, 획득한 확정 재화를 소모하여 캐릭터를 강화합니다.
* **확률형 리스크 시스템:** 레벨에 따른 차등 강화 확률을 적용하여 자원 관리에 긴장감을 부여합니다.

### ⚔️ Phase 3: 스테이지 보스전 (엔드 콘텐츠)
* **단계별 몬스터 객체:** 1스테이지(고블린)부터 10스테이지(드래곤)까지 난이도별 몬스터 객체가 구성되어 있습니다.
* **턴제 자동 전투:** '도전' 클릭 시 플레이어와 몬스터가 교대로 상대방의 HP-ATK 연산을 진행하여 데미지를 주고받는 함수가 실행됩니다.
